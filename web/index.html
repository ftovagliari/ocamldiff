OCamldiff

What is OCamldiff ?

OCamldiff is a small OCaml library providing functions to parse and display diff results.

OCamldiff was previously part of Cameleon but is now developped separately and is findlib compatible.

Download

OCamldiff is hosted on Github.

Releases

  • 1.1 [2013-06-03]: The Gtk interface is now distributed separately as Odiff-gtk.
  • 1.0 [2012-04-09]: first release after extraction from Cameleon.

Installation

To compile and install:

  • tar xvfz ocamldiff-XXX.tar.gz
  • cd ocamldiff-XXX
  • make all install
